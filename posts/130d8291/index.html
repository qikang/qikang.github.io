Docker Notepad

2020-04-19

  • docker images
    • 只查看 image 的tag
      docker images --format "{{.Repository}}:{{.Tag}}"
      docker images --format "table {{.ID}}\t{{.Repository}}:{{.Tag}}"
    • 查看 image 完整hash id
      docker images –no-trunc
    • 使用国内镜像地址下载 docker images。这里需要重启 docker 服务
    • 删除本地镜像
      docker rmi leslie/nginx:20170305.1
    • 构建镜像
      docker build -t leslie/nginx:20170305.1 .
  • docker ps 查看当前的容器
    • 查看所有容器
      docker ps -f "status=exited" --format={{.Names}}
      docker ps -f "status=running" --format={{.Names}}
